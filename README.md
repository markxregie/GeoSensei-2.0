##  About The Project
**GeoSensei** is a web-based, gamified geography learning platform designed to enhance spatial reasoning skills and user engagement. By integrating AI-driven feedback, crowdsourced geospatial data, and Mapillary street-level imagery, the platform presents real-world geographic information intuitively, making complex data easy to understand and interact with.

This project was developed between February 2025 and July 2025 as a full-stack initiative, focusing on clean UI/UX, responsive design, and robust data visualization.

## Key Features
* **Gamified Map Challenges:** Interactive challenges designed to test and improve users' spatial reasoning and geography skills.
* **AI-Driven Feedback:** Smart integration that provides users with dynamic, intelligent feedback based on their learning progress.
* **Real-World Imagery Integration:** Utilizes Mapillary to provide immersive, street-level geographic data.
* **Data Visualization Dashboard:** Transforms raw, complex geographic data into user-friendly, intuitive visual interfaces.

##  Tech Stack
* **Frontend:** JavaScript (React.js), HTML5, CSS3 
* **Backend:** Node.js
* **Map Services & APIs:** Mapillary
* **Database:** SQL 
* **Tools:** Git, GitHub, Figma (UI/UX Design)

## Research & Publication
The research and development behind this platform led to an official publication. The paper, titled *"Geosensei: A Web-based Learning Platform Integrating AI And Crowdsourced Geospatial Data For Gamified Geography Education,"* was presented and accepted at the **Nemesio Prudente International Research Conference**.
