## 📖 About The Project
**GeoSensei** is a web-based, gamified geography learning platform designed to enhance spatial reasoning skills and user engagement. By integrating AI-driven feedback and crowdsourced geospatial data, the platform presents real-world geographic information intuitively, making complex data easy to understand and interact with.

This project was developed between February 2025 and July 2025 as a full-stack initiative, focusing on clean UI/UX, responsive design, and robust data visualization.

##  Key Features
* **Gamified Map Challenges:** Interactive challenges designed to test and improve users' spatial reasoning and geography skills.
* **AI-Driven Feedback:** Smart integration that provides users with dynamic, intelligent feedback based on their learning progress.
* **Crowdsourced Geospatial Data:** Utilizes community-driven data to keep geographic information relevant and updated.
* **Data Visualization Dashboard:** Transforms raw, complex geographic data into user-friendly, intuitive visual interfaces.
